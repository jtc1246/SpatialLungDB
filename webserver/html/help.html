Tutorials

1. Introduction

Navigation bar is on the top, including home page, expression page, enrichment page, help page and about page. You can click the corresponding link to enter the page you want to visit.

Home page: The main page of the website, which provides a brief introduction of the website and the database, and a chatting box with our AI assistant.

Expression page: The page for you to visualize the expression of a single gene, or multiple genes. You can see the violin plot of a single gene, or the comparison of multiple genes in a dot plot.

Enrichment page: This page is to show the enrichment of several pathways, including beta cell pathways, cell death pathways, specific pathways.

Advanced enrichment analysis page: This page is to show the enrichment of customised pathways, which is defined by a set of genes.

There is also Help page and About page to show the tutorials and basic information of this website, and Small molecules page to show the information of small molecules.

2. Expression

This page is for you to visualize the expression of a single gene, or multiple genes. You can see the violin plot of a single gene, or the comparison of multiple genes in a dot plot. First you need to select whether you want to see one gene or multiple genes. If you want to see single gene, click "Gene", if you want to see multiple genes, click "Multiple Genes".

(1) Single gene

Input the gene id in the textbox, and then click "Search" button. After several seconds, the violin plot of the gene will be displayed.

Gene ids should look like INS, TOP2A, PDX1, AL691432.2, etc. (case insensitive) Note that not all gene ids have the expression (can generate the image) here. If that gene doesn't have the expression, it will show an error message.

(2) Multiple genes

You need to input the gene ids in the textbox, separated by space, comma, or new line, and then click "Submit" button. After several seconds, the dot plot of these genes will be displayed.

Gene ids should look like INS, TOP2A, PDX1, AL691432.2, etc. Here almost all gene ids can generate the image successfully, including the gene ids that don't have the expression.

3. Enrichment

This page is to show the enrichment plot of several pathways, including beta cell pathways, cell death pathways, specific pathways, and also customised pathways with your own gene list. First you need to select one of the three options (Beta cell pathways, Cell death pathways, Specific pathways) on the left side.

(1) Beta cell pathways

Click "Select a pathway" button, and then select a pathway from the dropdown menu. Then click "Submit", the enrichment of it will be displayed later.

(2) Cell death pathways

Click "Select a pathway" button, and then select a pathway from the dropdown menu. Then click "Submit", the enrichment of it will be displayed later.

(3) Specific pathways

Click "Select a pathway" button, and then select a pathway from the dropdown menu. You can also type in the textbox to search for the pathway name. Then click "Submit", the enrichment of it will be displayed later.

4. Advanced Enrichment Analysis

This page is to show the enrichment of customised pathways, which is defined by a set of genes.

Input the gene ids in the textbox, separated by space, comma, or new line, and then click "Launch" button. Gene ids should look like INS, TOP2A, PDX1, AL691432.2, etc. (case insensitive) After several seconds, the enrichment of these genes will be displayed.
